uatkan website PHP untuk digital agency "dopagency" dengan struktur multi-page menggunakan PHP includes.

**REQUIREMENTS:**

**1. STRUKTUR HALAMAN:**
- index.php (Home)
- about.php (About)
- service.php (Service) 
- organisasi.php (Organisasi)
- contact.php (Contact)

**3. COLOR PALETTE:**
- Primary: #1a47fb
- Accent: #00d9ff
- Background: #20bfe

**4. ASSET:**
- Logo sudah tersedia di assets/logo/logo.png
- Portfolio images sudah tersedia di assets/portfolio/

**5. SETIAP HALAMAN PHP HARUS:**
- Define $page_title dan $active_page di awal
- Include header.php, navbar.php, footer.php
- Responsive design
- Consistent navigation dengan active state

**6. SECTIONS PER HALAMAN:**

**index.php:**
- Hero section
- Services preview (4 cards)
- Portfolio showcase
- Testimonials
- CTA section

**about.php:**
- About hero
- Agency story
- Team showcase
- Stats/achievements

**service.php:**
- Services hero
- 6 detailed services dengan icon
- Service process
- Why choose us

**organisasi.php:**
- Organizational structure
- Team hierarchy
- Company culture
- Values & mission

**contact.php:**
- Contact form (Name, Email, Subject, Message)
- Contact info
- Social media
- Google Maps (optional)
